# Chrome Web Store Test Instructions

No login is required.

## Reviewer Flow

1. Install the extension.
2. Click the extension icon and pin it if needed.
3. Open the popup and turn Reading mode on.
4. Visit any English webpage, for example a news article.
5. Select one short sentence or paragraph.
6. Wait for the vocabulary bubble to appear near the selected text.
7. Click `Save word` on any card.
8. Open the popup again and click `Saved words & review`.
9. Confirm that the saved word appears in the list.
10. Open `Flashcard review` and click `Show meaning`.

## Expected Result

- The bubble shows only vocabulary at or above the selected CEFR level.
- The extension stays on the same page while the user reads.
- Saved words remain available in the dashboard and review mode.
