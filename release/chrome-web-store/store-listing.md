# CEFR Reading Assistant Store Listing

## Short Description

Understand difficult English words on webpages without leaving the page.

## Full Description

CEFR Reading Assistant helps English learners keep reading.

Select text on a webpage and the extension shows only the words that are difficult for your CEFR level. Each word appears in a small bubble near the text, with a simple English meaning and a short example sentence.

Use it for:

- Reading news and articles without opening another tab
- Focusing only on vocabulary that matches your level
- Understanding words in the current context
- Saving useful words for later review

Main features:

- CEFR level setting from A1 to C2
- Reading mode toggle
- Automatic analysis when you select text
- Right-click analysis
- Bubble cards near the selected text
- Simple English meanings and examples
- Local saved word bank
- Flashcard review mode with random order

Privacy notes:

- The extension analyzes only the text you select
- Saved words and settings are stored locally in Chrome
- Selected text may be sent to the extension backend and AI provider to generate meanings and examples

## Single Purpose

Help English learners understand difficult words in selected webpage text by showing simple inline vocabulary support.
